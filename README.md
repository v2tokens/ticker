# ticker

> **Work In Progress**

Ticker driving code.

## setup

```bash
$ git clone https://github.com/v2tokens/ticker && cd ticker
$ python3 -m venv .venv && source .venv/bin/activate
$ make
```

## run it

> **Important: the script must be run as root**

```bash
$ source .venv/bin/activate
$ python3 app.py
```
